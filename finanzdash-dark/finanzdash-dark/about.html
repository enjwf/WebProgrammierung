Über & Dokumentation

Diese Website erfüllt die Aufgaben-Vorgaben:

  • Navigation mit ≥3 Unterseiten: Start, Aktien, Krypto, Über.
  • Dokumentation: Installation & Funktionsbeschreibung (unten).
  • Sauberes Coding: Modularer Code, Kommentare, getrennte Styles/JS, Framework-Einbindung.
  • Mind. zwei API-Endpoints: Financial Modeling Prep /search, /quote, /historical-price-full; CoinGecko /market_chart, /markets.
  • Sinnvolle Darstellung: Karten, Badges, Loader, Charts (Chart.js).
  • Responsive Layout: Bootstrap 5, mobile-first.

Installation

  1. Ordner in VS Code öffnen.
  2. Extension Live Server starten und index.html öffnen.

Funktionen

  • Aktien: Suche mit Autocomplete (FMP /search), Detaildaten (FMP /quote), Verlaufs-Chart (FMP /historical-price-full), Vergleich von bis zu 3 Tickern, Watchlist (LocalStorage).
  • Krypto: Preis-Chart und Kennzahlen (CoinGecko /market_chart, /markets).
  • Start: Quick-Search, Beispiel-Widgets (AAPL, BTC) und Watchlist-Übersicht.
